# BrokerFlow AI

BrokerFlow AI is a knowledge and action stack for building loan broker automation assistants.

## What This Repo Contains

- GPT knowledge files
- Custom GPT Action documentation
- No-auth webhook action ideas
- API key and OAuth action patterns
- OpenAPI schema examples
- n8n, Zapier, and Make workflow docs
- Broker SOPs
- Email/SMS templates
- CRM and pipeline field dictionaries
